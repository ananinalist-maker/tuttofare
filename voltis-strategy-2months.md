### 1.2 Что работает

- **CPL €2.12** в мае — отличный показатель для премиум-продукта
- **Скандинавия + Бенилюкс** — лучшие гео (NO, SE, DK, BE, NL)
- Оптимизация **LEAD_GENERATION** + простая форма = лучшая связка
- Статичные креативы дают CTR 5.19% — выше рынка
- Продукт уникален: ни один конкурент не даёт Player Report + план 6 мес. + 30 дней поддержки

### 1.3 Ключевые проблемы

| # | Проблема | Влияние |
|---|---|---|
| 1 | **Нет воронки прогрева** — холодный трафик ведётся сразу на продажу €2,500 | Конверсия 0.4% вместо 5-8% |
| 2 | **Instagram @voltis.camp не привязан** к рекламному кабинету | Реклама в IG идёт без аккаунта, теряются подписки |
| 3 | **0 кастомных аудиторий** в кабинете | Нет ретаргетинга, нет LAL |
| 4 | **Нет email-маркетинга** | 162 качественных лида не прогреваются |
| 5 | **50+ адсетов за 50 дней** — хаотичная структура | Бюджет размазан, алгоритм не обучается |
| 6 | **Нет системы обработки лидов** | Медленные ответы, нет follow-up, нет скрипта |

### 1.4 Главный вывод

> **Реклама работает. Проблема — в конверсии после лида.**
> При текущих 0.4% для 10 продаж нужен бюджет €7,800. При конверсии 5% (воронка + прогрев) — достаточно €1,500–2,000.

## 2. СТРАТЕГИЯ: ОТ ЛИДА К ПРОДАЖЕ ЗА 4 КАСАНИЯ

### Текущая модель (не работает)

```
Холодный трафик → Лид-форма → Менеджер звонит → "Дорого" / "Нет ответа"
(1 касание до продажи)
```

### Новая модель

```
КАСАНИЕ 1: Контент-реклама (видео/карусель) → человек смотрит, запоминает
КАСАНИЕ 2: Ретаргетинг с соц. доказательством → заполняет лид-форму
КАСАНИЕ 3: Менеджер + email-серия + WhatsApp → доверие, ответы на вопросы
КАСАНИЕ 4: Оффер Early Bird + дедлайн + звонок → ПРОДАЖА
```

**Почему это работает:** родитель не отдаёт €2,500 незнакомому лагерю с первого касания. Нужно 4–7 точек контакта: увидел → заинтересовался → доверился → купил.
